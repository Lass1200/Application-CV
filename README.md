APPLICATION CV – REACT (entrainement)

Description :
Cette application permet de créer un CV dynamiquement avec React.
L’utilisateur peut ajouter, modifier et afficher ses informations personnelles,
formations et expériences professionnelles.

--------------------------------------------------

DEMO EN LIGNE
Application déployée sur Vercel :


--------------------------------------------------

FONCTIONNALITÉS

- Ajout et modification des informations générales
- Ajout de plusieurs formations
- Ajout de plusieurs expériences professionnelles
- Aperçu du CV en temps réel
- Formulaire et aperçu du CV affichés côte à côte

--------------------------------------------------

TECHNOLOGIES UTILISÉES

- React
- Hook useState
- CSS (Flexbox)
- Git & GitHub
- Vercel pour le déploiement

--------------------------------------------------


COMPÉTENCES DÉVELOPPÉES

- Gestion du state avec React
- Lift State Up
- Manipulation de tableaux avec map()
- Mise à jour d’un élément spécifique dans un tableau
- Rendu conditionnel
- Déploiement d’une application React
